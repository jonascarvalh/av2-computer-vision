## Como Executar?
Crie  e ative um ambiente virtual:
```sh
py -m venv venv
cd ./venv/Scripts/activate
```

Instale as dependências no ambiente virtual:
```sh
pip install -r requirements.txt
```

Agora basta abrir o jupyter:
```sh
jupyter notebook
```
